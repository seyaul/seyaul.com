Step 5 of 6
Configure trigger A

Set up the first automation in the Shortcuts app. Follow these steps exactly:

  1. Open Shortcuts → tap Automation at the bottom
  2. Tap + → App
  3. Tap Choose, select the study app → tap Opens
  4. Tap Next → Add Action
  5. Search for Scriptable → select Run Script
  6. Under Script, choose InstagramStart
  7. Turn off Ask Before Running → tap Done
6
Step 6 of 6
Configure trigger B

Repeat the process for the closing automation:

  1. Open Shortcuts → Automation → + → App
  2. Select the same study app → tap Closes
  3. Tap Next → Add Action → search Scriptable → Run Script
  4. Under Script, choose InstagramStop
  5. Turn off Ask Before Running → tap Done
✓

Setup complete

Your device is now configured for the study. The system will run automatically in the background. You can close this page.
